# 🏛️ Coliseu Academia — App PWA

## Como publicar GRÁTIS em 10 minutos

---

### 📁 Arquivos necessários (todos na mesma pasta)
- `index.html`
- `manifest.json`
- `sw.js`
- `icon-192.png`
- `icon-512.png`

---

### PASSO 1 — Criar conta no GitHub
1. Acesse **github.com**
2. Clique em **Sign up**
3. Crie sua conta (gratuita)

---

### PASSO 2 — Criar repositório
1. Clique no **+** no canto superior direito
2. Clique em **New repository**
3. Nome: `coliseu-app`
4. Marque **Public**
5. Clique em **Create repository**

---

### PASSO 3 — Enviar os arquivos
1. Na página do repositório, clique em **uploading an existing file**
2. Arraste todos os 5 arquivos para a área
3. Clique em **Commit changes**

---

### PASSO 4 — Ativar o GitHub Pages
1. Clique em **Settings** (no repositório)
2. No menu lateral, clique em **Pages**
3. Em "Branch", selecione **main**
4. Clique em **Save**
5. Aguarde ~2 minutos

---

### PASSO 5 — Seu link estará disponível
```
https://SEU_USUARIO.github.io/coliseu-app/
```

✅ **Pronto! Mande esse link para os alunos.**

---

### 📱 Como instalar no celular (Android)
1. Abra o link no **Chrome**
2. Vai aparecer um banner: **"Adicionar à tela inicial"**
3. Toque em **Instalar**
4. O app aparece na tela como qualquer outro app

### 🍎 Como instalar no iPhone
1. Abra o link no **Safari**
2. Toque no ícone de **compartilhar** (quadrado com seta)
3. Toque em **Adicionar à Tela de Início**
4. Toque em **Adicionar**

---

### 🔄 Como atualizar o app
1. Edite o `index.html` no seu computador
2. Acesse **github.com/SEU_USUARIO/coliseu-app**
3. Clique no arquivo `index.html`
4. Clique no ícone de **lápis** (editar)
5. Cole o novo conteúdo
6. Clique em **Commit changes**
7. Em ~2 minutos todos os celulares recebem a atualização automaticamente ✅
